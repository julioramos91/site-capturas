Política de Privacidade

Atualizado em

Resumo

Este aplicativo solicita permissões do Google para ler emails em modo somente leitura e registrar resultados em planilhas do próprio usuário. Não mantemos servidores próprios e não compartilhamos dados com terceiros.

Dados coletados/tratados

  • Metadados de mensagens que correspondem aos filtros (remetente, destinatário, assunto, data/hora).
  • Trechos estritamente necessários para identificar notificações e valores (ex.: saldos/códigos) a partir do corpo do email.
  • Não é feita cópia integral de mensagens; não acessamos rascunhos, spam ou lixeira.

Finalidade

Organizar e registrar, em Google Sheets do próprio usuário, ocorrências como saldos, cupons e avisos, de forma automatizada para consulta pessoal.

Armazenamento e retenção

  • Os resultados são gravados exclusivamente em planilhas do Google na conta do usuário.
  • Não copiamos dados para servidores externos.
  • A retenção é controlada pelo usuário (pode editar/apagar linhas a qualquer momento).

Compartilhamento

Não há compartilhamento de dados com terceiros, nem uso para publicidade.

Segurança e escopos solicitados

O acesso ocorre via APIs Google com os escopos mínimos necessários:

  • https://www.googleapis.com/auth/gmail.readonly — ler mensagens relevantes para extração.
  • https://www.googleapis.com/auth/spreadsheets — escrever resultados na planilha do usuário.
  • https://www.googleapis.com/auth/userinfo.email — identificar a conta que executa.
  • https://www.googleapis.com/auth/script.scriptapp — criar/gerenciar gatilhos de execução.
  • https://www.googleapis.com/auth/script.container.ui — exibir painel/menus no Google Sheets.

Controle do usuário

  • Revogar acesso: myaccount.google.com/permissions.
  • Exclusão de dados: apagar diretamente as linhas na planilha vinculada.

Conformidade com a política do Google

Este app segue a Google API Services User Data Policy, incluindo os requisitos de Limited Use para dados do Gmail: uso limitado à funcionalidade do app, sem transferência a terceiros e sem uso para anúncios.
